# Teoria da Computação — Sprint Planning como Knapsack

O objetivo da atividade é comparar diferentes abordagens para resolver o problema de otimização de Sprint Planning utilizando conceitos inspirados no problema da Mochila (Knapsack).

## Integrantes

- Luiz Felipe Schroder Marcon

- Rebeca Lara de Souza

## Exercícios Implementados

- Exercício 1 — Busca Exaustiva (Brute Force)
- Exercício 2 — Heurística Gulosa (Greedy)
- Exercício 3 — Análise Empírica de Complexidade
- Exercício 4 — Hill Climbing

## Tecnologias Utilizadas

- Python 3

## Como Executar

Clone o repositório:

```bash
git clone URL_DO_REPOSITORIO
```

Acesse a pasta:

```bash
cd NOME_DO_REPOSITORIO
```

Execute o arquivo:

```bash
python exercicios_aula1.py
```

## Objetivo da Atividade

Comparar abordagens exatas, heurísticas e busca local para resolução do problema de seleção de tarefas em Sprint Planning, analisando desempenho, qualidade da solução e complexidade computacional.
